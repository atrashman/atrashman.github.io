对自己任给的一个整数，求出其各位数字，并生成元组。
这次我们定义一个函数，直接用return语句，也算是一行代码解决了

def kk(num):
    return tuple(map(lambda x:int(x),str(num)));

Python的内置函数和列表的方法中都没有提供求中位数的函数。对任意输入的数值列表（自己给定），输出这个序列的中位数。

def middle_num(num_list):
    return len(num_list)%2==0 and \
    (sorted(num_list)[len(num_list)//2-1]+sorted(num_list)[len(num_list)//2-1])/2 or \
    sorted(num_list)[len(num_list//2)]
    #小小测试一下
middle_num([1,23,21,2,31,12])
